背后的按键可以切换运行模式
1.测温上传 -> 2.setup设置 -> off 关机

设置温度计wifi上网的方法:

如果wifi有wps快速配网功能，按一下无线AP的WPS键，
然后插上usb线，切换到setup模式,就会自动进行WPS配网，
如果WPS配网没有成功，进入smartconfig模式。 温度计显示 CO xx, 
此时手机连上家里2.4G的wifi网络， 用微信浏览网址
http://wx.ai-thinker.com/api/old/wifi/config
就可以给附近的温度计配置wifi上网密码。配置完成后，温度计会自动重启。
如果上述方法没有成功， 那么可以试试用温度计的AP服务，
在setup模式下, 温度计会生成一个空密码的名字叫disp的wifi接入点AP, 用手机或者笔记本可以搜到它
用手机(先关掉4G)或电脑，连上无线接入点disp后，用浏览器浏览网页http://192.168.4.1，
会打开web设置界面，把家里的WiFi和密码设置进去，然后保存重启。

远程显示温度方式:
温度历史数据地址1:https://temp2.wf163.com:1443/disp

Q&A:
Q:为啥会显示[ 3.xx ]或者[ 4.xx ]，并闪动?
A:这是温度计没有连上网络，它会显示电池电压, 需要给温度计配上wifi，让它上网。

Q:为啥温度计显示[OFF3.x]?
A:电池电压过低， 3.x是电池电压, 插上usb线充电就可以了。

Q:为啥显示温度会偏高?
A:在配置wifi时， 可能会因为温度计持续工作， 而造成温度升高， 等1小时后，它下次自动测量数据，就可以显示准确的气温，误差0.5度

Q:还有更详细的说明吗?
A:看源码网站:https:/github.com/lshw/wifi_disp

Q:怎么可以知道设备的序列号?
A:温度计上会有贴纸，写着序列号,如果找不到贴纸， 可以用以下方法， 找到序列号:
A:配好温度计的wifi设置后， 按背后的按键， 让它进入OTA模式, 等它完成上网， 就会在屏幕显示ip地址， 用手机或者电脑浏览这个ip地址，
   出来的设置页面上，可以显示温度计序列号
